# ESP32_OLED_METEOSOURCE
  
1.3" oled display 128x64. sh1106 controller.  
ESP32-C3-SuperMini  

3.3V -- VCC  
GND  -- GND  
D5   -- SCL  
D6   -- SDA  
  
It need "config.txt" with requiring following info.  
ssid=(wifi name)  
pass=(wifi pass)  
lat=(your location)  
lon=(your location)  
key=(meteosource appid)  
timezone=(timezone delta in hours)
